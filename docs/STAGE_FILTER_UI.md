## Key UI Changes

1. **Filter Layout**: Changed from single column to 2-column grid on medium+ screens
2. **New Control**: Added multi-select dropdown for Stage filtering
3. **Dynamic Count**: Row count updates in real-time based on filter
4. **Visual Feedback**: Selected stages shown in dropdown label
5. **Disabled State**: Filter is disabled until stage data loads
